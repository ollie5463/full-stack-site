## Basic-site-BE
This is to be used in conjunction with [Basic-site-FE](https://github.com/ollie5463/basic-site-FE).

### How to use
Prerequisites
- Have node_modules installed by using the `yarn` command via the CLI

To start this project you need to run:
```js
yarn start
```

### Tech used
- Node (path, fs)
- Yarn
- Express
